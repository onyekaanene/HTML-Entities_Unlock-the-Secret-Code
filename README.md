# HTML Entities: Unlock the Secret Code

The quick, no-nonsense guide to displaying special characters in HTML without breaking your page.

## What It Covers

- Why characters like `<`, `>`, `&`, and `"` need special handling in HTML
- How HTML entities work — the `&` to `;` syntax explained
- Named vs. numeric entities: when to use each
- Practical code examples for common symbols (copyright, trademark, punctuation)
- A ready-to-use reference table of the most essential entities

## Why It Matters

Special characters are everywhere — legal copy, pricing, code snippets, multilingual content. A developer who reaches for `&copy;` instead of pasting `©` directly, or who knows why `&amp;` exists, writes HTML that is robust across browsers and encodings. This tutorial reflects an understanding of how browsers parse markup, not just how to make it display correctly.

## Quick Reference Table

| Character | Entity | Description |
|---|---|---|
| `<` | `&lt;` | Less-than |
| `>` | `&gt;` | Greater-than |
| `&` | `&amp;` | Ampersand |
| `"` | `&quot;` | Double quote |
